# 🍳 Kitchen P&L Analysis Dashboard

An interactive, live financial dashboard built with **Python** and **Streamlit** to track, analyze, and visualize Profit and Loss metrics for kitchen operations.

### 🔗 [Live Demo](https://kitchen-p-l-analysis-jxrzh3cijsktatgmcfkjqx.streamlit.app/)

---

## 📊 Business Overview
This project transforms raw financial data into actionable insights for kitchen management. It provides a granular view of financial health, helping stakeholders identify high-cost areas and optimize profit margins.

### Key Features:
*   **Real-time P&L Tracking:** Instant calculation of Gross Profit, Operating Expenses, and Net Profit.
*   **Cost Analysis:** Deep dive into COGS (Cost of Goods Sold) and labor costs.
*   **Interactive Visualizations:** Trend lines for revenue and bar charts for expense distribution.
*   **Dynamic Filtering:** Filter data by date ranges or specific categories to isolate performance trends.

## 🛠️ Tech Stack
*   **Language:** Python 3.x
*   **Frontend/Hosting:** Streamlit
*   **Data Manipulation:** Pandas, NumPy
*   **Data Visualization:** Plotly / Matplotlib
